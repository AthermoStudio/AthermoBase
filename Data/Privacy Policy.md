## 2. Legal Basis for Data Processing

We process user data based on the following legal grounds under the GDPR:

* Performance of a contract – necessary for account creation, login, and providing AthermoBase services.
* Legitimate interest – for ensuring security, preventing fraud, and protecting the service.
* Consent – where required (e.g., optional features such as location data, if enabled in the future).

---

## 3. What data do we collect?

As part of using AthermoBase, the following data may be processed:

* username (nickname),
* email address (if the user chooses to provide it),
* hashed user passwords,
* hashed account recovery codes,
* data related to two-factor authentication (2FA/TOTP),
* session tokens and data necessary to maintain active sessions,
* information about application usage time,
* information about achievements, progress, and user profile data,
* information about user ranks,
* user IP address,
* approximate geographic location (derived from IP address, country/region level; not GPS-based)

---

## 4. Purpose of Data Processing

User data is processed for the following purposes:

* enabling account registration and login,
* maintaining the security of user accounts,
* supporting two-factor authentication (2FA),
* storing user progress,
* storing achievements and statistics,
* recovering account access,
* detecting abuse, hacking attempts, and security breaches,
* maintaining and developing the AthermoBase service.

---

## 5. Data Retention Period

User data is retained for the duration of the account's existence.

Information about active sessions is retained until they expire or are manually deleted by the user.

Account security data may be retained for the period necessary to ensure the security of the service.

After deletion of the account, user data is deleted or anonymized, with the exception of data whose continued retention is required by applicable law.

---

## 6. User Rights

Users have the right to:

* access their data,
* correct their data,
* delete their data,
* restrict the processing of their data,
* lodge a complaint with the competent supervisory authority.

---

## 7. Data Sharing

User data is not sold or made available to third parties, except as required by applicable law.

---

## 8. Data Security

AthermoBase employs technical and organizational measures to protect user data against unauthorized access, loss, modification, or disclosure.

User passwords and account recovery codes are stored as cryptographic hashes.

Users can further secure their accounts by enabling two-factor authentication (2FA).

---

## 9. Third-Party Services

To operate AthermoBase, we may use third-party services such as:

* hosting and server infrastructure providers,
* email service providers,
* authentication or security services.

These providers may process limited user data strictly for the purpose of service operation and under data processing agreements.

---

## 10. Technical Data Storage

The application may store local session data, such as authentication tokens, to maintain user login state.

These tokens are stored securely and are required for proper functioning of the service.

The Privacy Policy may be updated as the AthermoBase project develops.

Users can be notified of significant changes via the launcher or other communication channels related to the service.

The current version of the document is always available with the AthermoBase service.
